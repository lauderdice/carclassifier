Tato aplikace slouží k rozpoznávání aut dle obrázků. Na MS Azure je deploynuté API, které komunikuje s JS aplikací a posílá jí odpovědi. Iniciální stránka obsahuje validátor správného formátu obrázků, je použito History API, prvky z JQuery, LocalStorage API. Na homepage je rovněž použito Media API ve formátu hudebního videa s Michaelem Schumacherem.
